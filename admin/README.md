# TransCars Admin

Site statique pour gerer les tournees depuis un ordinateur.

## Render Static Site

- Repository : `ThomasMat85/Application-Transporteur`
- Root Directory : `admin`
- Build Command : laisser vide
- Publish Directory : `.`

Le site appelle l'API publique :

```text
https://application-transporteur.onrender.com
```

L'API accepte les sites Render en `https://*.onrender.com` via CORS.
